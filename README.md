# 📱 Perfect Notes - Aplikasi Catatan Android Sempurna

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Kotlin](https://img.shields.io/badge/kotlin-1.9.20-orange.svg)
![API](https://img.shields.io/badge/API-24%2B-brightgreen.svg)

Aplikasi catatan Android dengan arsitektur MVVM dan fitur lengkap untuk produktivitas sehari-hari.

## ✨ Fitur Utama
- ✅ **CRUD Lengkap** - Buat, baca, update, hapus catatan
- ✅ **Arsip & Pin** - Atur catatan penting dan jarang dipakai
- ✅ **Kategori & Pencarian** - Organisasi dan temukan catatan dengan mudah
- ✅ **Reminder** - Pengingat untuk catatan penting
- ✅ **Lampiran Gambar** - Tambahkan gambar ke catatan
- ✅ **Kunci Biometric** - Lindungi catatan sensitif
- ✅ **Dark Mode** - Tema gelap untuk kenyamanan mata
- ✅ **Backup & Restore** - Cadangkan ke cloud atau lokal
- ✅ **Markdown Support** - Format teks kaya
- ✅ **Widget** - Catatan cepat dari home screen

## 🏗️ Arsitektur
- **MVVM** (Model-View-ViewModel)
- **Clean Architecture** dengan 3 layer (Data, Domain, Presentation)
- **Repository Pattern**
- **Dependency Injection** (Hilt)

## 🛠️ Teknologi
- **Kotlin** - Bahasa pemrograman utama
- **Android Jetpack**:
  - Room Database
  - ViewModel & LiveData
  - Navigation Component
  - WorkManager
  - Preferences
- **Coroutines & Flow** - Asynchronous programming
- **Hilt** - Dependency Injection
- **Glide** - Image loading
- **Material Design 3** - UI/UX modern

## 📱 Minimum Requirement
- Android 7.0 (API 24) ke atas
- RAM minimal 2GB
- Ruang penyimpanan 50MB
